Unlocking IBM Bob: A Deep Dive and Showcase

About the Event

Join us for an exclusive, developer-focused session on IBM Bob—IBM's next-generation agentic AI coding partner. Built to transcend basic autocomplete utilities, IBM Bob represents a major leap in operationalizing AI across the entire software development lifecycle. In this meetup, we will take a deep dive into Bob's capabilities, demonstrating how it acts as an autonomous yet human-in-the-loop developer assistant.

What We'll Cover:

  • Agentic vs. Autocomplete: Understanding the architecture of agentic coding assistants and how IBM Bob operates as a true development partner.
  • Actionable Capabilities Showcase: Watch Bob read codebases, run terminal commands, and invoke Model Context Protocol (MCP) tools live.
  • Enterprise Security & Compliance: A look at how Bob complies with strict security frameworks (HIPAA, FedRAMP) while maintaining high speed and flexibility.
  • Tailored Development Modes: How to utilize specialized configurations (like Review Mode) to accelerate debugging, testing, and modernization tasks.
